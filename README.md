# HandsOn Kafka with Elastic


## Create Enviroment Pynton

```shell
python3 -m venv .venv
```

## Activate Enviroment Pynton

```shell
. .venv/bin/activate
```
## Update Enviroment Pynton

```shell
python3 -m pip install --upgrade pip
```

## Requirements

```shell
pip install Elasticsearch
```

## Run project

```shell
python src/main.py
```
